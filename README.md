Beauty Salon CRM System 
Professional Management System for Beauty Salons — це консольна CRM-система, побудована на архітектурі Python + PostgreSQL. Проєкт реалізовано з чітким розділенням бізнес-логіки та роботи з даними, що забезпечує легку підтримку та масштабованість.

Основні можливості
Система розділена на 3 основні ролі з унікальним функціоналом:

Адміністратор:

Керування персоналом (додавання/видалення майстрів).

Керування прайс-листом послуг.

Модерація записів (підтвердження або скасування візитів).

Майстер:

Перегляд власного розкладу.

Перегляд призначених клієнтів та послуг.

Клієнт:

Зручний пошук майстра за послугою.

Перегляд вільного часу в реальному часі.

Швидкий запис на процедуру.

Технологічний стек
Мова: Python 3.12

База даних: PostgreSQL

Драйвер БД: psycopg2

Безпека: werkzeug (хешування паролів за стандартом PBKDF2).

Оточення: python-dotenv (захист конфіденційних даних бази).

Архітектура проєкту
Проєкт реалізовано за принципом Repository Pattern:

database.py: Ініціалізація підключення та створення структури таблиць.

repositories.py: Шар абстракції для SQL-запитів (Users, Masters, Services, Bookings, Schedule).

admin.py, master.py, client.py: Менеджери бізнес-логіки для кожної ролі.

main.py: Точка входу з системою автентифікації.
